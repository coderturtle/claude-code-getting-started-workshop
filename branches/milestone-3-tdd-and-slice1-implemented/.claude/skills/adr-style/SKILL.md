---
name: adr-style
description: Enforces ADR markdown format with YAML frontmatter and standard sections. Use whenever creating or editing ADRs.
---

## Filename
- `ADR-XXXX-short-title.md`

## Frontmatter (required)
- `id`, `title`, `status`, `date`

## Sections (required)
- Context
- Decision
- Consequences
- Alternatives

## Tone
Concise, explicit tradeoffs, link to relevant docs or PRs when helpful.
